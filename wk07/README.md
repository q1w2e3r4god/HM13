# HM13 크리스마스 카드 프로젝트 

## 파일구조 및 설명
* hm13_card.html : 크리스마스 트리, 장식품, 음악 등을 position을 이용하여 테두리 이미지 안쪽의 적절한 곳에 위치시킴과 animation과 transform을 적극 활용하여 장식들을 꾸밈
* media : 사진, 음악 등의 자료 보관

## 사용한 html, css 기술 
* audio
* img src
* border-radius
* position
* animation
* keyframes
* transform
* scale
* hover
* hr
* br
